# 다다름(Dadareum) 팀장 리더십 역량교육 과정

다다름의 대표 프로그램인 **팀장 리더십 역량교육 과정**을 위한 프리미엄 랜딩 페이지입니다. 본 프로젝트는 학습자의 자기인식과 행동변화를 목적으로 설계된 DC(Development Center) 기반의 교육 프로그램을 소개합니다.

## 🚀 주요 특징

- **Premium Design**: 고해상도 배경 이미지와 블러 효과, 세련된 컬러 팔레트를 사용한 현대적인 웹 디자인
- **Fixed Navigation**: 스크롤 시 상단에 고정되는 반응형 메뉴 바
- **Interactive Elements**:
  - 수치 애니메이션 (Score bar animation)
  - Intersection Observer를 활용한 요소별 페이드인(Fade-in) 효과
  - 프리미엄 문의하기 모달 (회사명, 담당자명, 연락처 입력 기능)
- **Typography**: Pretendard 폰트를 적용하여 높은 가독성 확보
- **Responsive Layout**: PC와 모바일 환경 모두에 최적화된 반응형 레이아웃

## 🛠 기술 스택

- **Frontend**: HTML5, Vanilla CSS, Vanilla JavaScript
- **Fonts**: Pretendard (CDN)
- **Deployment**: GitHub Pages 또는 정적 서버 호스팅 가능

## 💻 로컬에서 실행하기

이 프로젝트는 별도의 빌드 과정 없이 정적 파일로 구성되어 있습니다. `serve` 패키지를 사용하여 로컬에서 바로 확인할 수 있습니다.

```bash
# npx를 사용하여 실행
npx serve .
```

실행 후 기본적으로 `http://localhost:3000` (또는 지정된 포트)에서 확인 가능합니다.
